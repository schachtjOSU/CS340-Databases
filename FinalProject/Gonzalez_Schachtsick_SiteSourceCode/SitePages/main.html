Homebrew Database

By Saul Gonzalez and Jeffrey Schachtsick

A homebrew store is a store for 'do-it-yourselfers' which carries such things as ingredients and hops in crafting beer at home. This database will simulate an inventory system where a user can add items to the database and make updates where needed.

Table .................... What you can do

Beer ...................... View and Add Beers

Recipe ................... View and Add Recipes

Ingredient .............. View, Add and Update Ingredients

Recipe_Ingredient . View and Add Quantities of Ingredients to Recipe

Hops ....................... View and Add Hops
